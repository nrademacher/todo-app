This is a simple, light-weight boilerplate for getting you started quickly with your next #webdev 👩🏻‍💻 project.

The idea is to enable you to dive right in on a project idea with zero time spent on config, yet with all the essential #DX 👌🏽 features, such as hot reloading and optimized production builds.

To get started, simply start editing this index.html file, while using #TailwindCSS to add some breezy 💅🏿 gloss.

You don't have to use any JS framework/library, but if you fancy #ReactJS , you can turn this boilerplate into a React app by running yarn setup:react.

Now go and build something. 💪
